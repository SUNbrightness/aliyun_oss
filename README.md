# aliyun_oss_flutter

aliyun oss plugin for flutter. Use this plugin to upload、download、list files to aliyun oss.

## Usage

## 功能
- [x] PutObject 添加的Object大小不能超过5 GB。
- [x] ListObject 仅仅获取1000数据量一下的文件，oss设定
- [x] GetObject 下载文件到本地
- [] ListObject 分页获取文件
